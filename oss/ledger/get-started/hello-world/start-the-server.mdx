# Start the Numary Ledger service

:::info
Numary Ledger is a service that must be running before it can process any transactions.
:::

To start the Numary Ledger service, enter the following into your command line:

```shell
numary server start --verbose
```

If all is well, you should see something like this in your shell:
```
Version: 1.0.0-rc3 
Date: 2022-01-27T10:53:14Z 
Commit: 266c79e 
running... 
```

:::caution

Seeing something else instead? Head to the troubleshooting section to get it sorted.

:::
