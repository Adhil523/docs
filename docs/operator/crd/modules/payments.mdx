---
title: Payments
---

Formance Payments is a unified API that abstracts over multiple different payment providers like Stripe and Wise, simplifying the process of building money flows that stitch multiple payment providers together.

## Requirements

Formance Payments requires:
- **PostgreSQL**: See configuration guide [here](/next/operator/infra/postgres).
- (Optional) **NATS**: See configuration guide [here](/next/operator/infra/nats).

## Payments Object

:::info
You can find all the available parameters in [the comprehensive CRD documentation](/next/operator/crd#gateway).
:::

```yaml
apiVersion: formance.com/v1beta1
kind: Payments
metadata:
  name: formance-dev
spec:
  stack: formance-dev
```
