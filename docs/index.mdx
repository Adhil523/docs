Formance Stack is an open-source suite of modular tools designed to help developers build, operate, and track money movements of any size and shape. The suite includes a range of products that provide technical introductions, how-tos, and reference documents for each component. Whether you are building a small money-moving application or a large-scale financial platform, Formance Stack has the tools and resources you need to get started:

<DocCardGrid>

<DocCard headline="Ledger" cta="Learn More" link="/ledger/" highlight="true">
  Model and record complex money movements. Accurately track revenue and account balances across your entire operation.
</DocCard>

<DocCard headline="Payments" cta="Learn More" link="/payments/" highlight="true">
  Unified data layer and API for payment processing services like Stripe and Wise, with upcoming support for payment initiation.
</DocCard>

<DocCard headline="Wallets" cta="Learn More" link="/wallets/" highlight="true">
  Manage and control user funds in customizable white-label wallets with built in rules and flows.
</DocCard>
